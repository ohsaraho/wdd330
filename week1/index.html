Notes

  • JavaScript Local Storage uses "key-value-pairs" meaning the key would be zipcode and the value would be 85504 for example
  • You use "setItem()" with localStorage as setItem(key, value) to store the value in the object
  • You use "getItem()" as setItem(key) to get the value from the object it was stored in
  • Use "value" when wanting to get the user's information from the inputs or textarea
  • Use "innerHTML" when using a section tag for example
